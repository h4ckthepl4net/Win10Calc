Class: ValueSelectButton

Components.ValueSelectButton()

Class for 'value-select-button' custom element. 'value-select-button' is used to select one value from a list. When it's inner button is clicked 'value-select-button' changes selected value to next value in array which is specified in value-list attribute. To see all non-overriden static properties and methods see BaseComponent
